## 🐛 Troubleshooting

**Gallery password not working?**
- Check: zodiac sign spelled correctly
- Check: mother's name spelled correctly
- Check: lowercase spelling
- Check: no spaces between parts

**Links showing 404?**
- Check file names (case-sensitive)
- Use relative paths: `projects.html` not `/projects.html`
- Verify files are in same directory

**Styling looks broken?**
- Clear browser cache (Ctrl+Shift+Delete)
- Verify styles.css is loaded (check DevTools > Network)
- Check file sizes are correct

**Games links not working?**
- Check internet connection (external sites)
- Verify URLs are current
- Use alternative game sites if needed
