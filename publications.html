2023

Real Time Detection of Regime Shifts in Multivariate Data Streams

Adnan Bashir, Trilce Estrada, et al.

DAWAK 2023 | International Conference on Data Warehousing and Knowledge Discovery

This paper presents a novel approach to detecting regime shifts in high-dimensional streaming data using unsupervised learning techniques. We introduce DRUM (Distributed Real-time Unsupervised Modeling), a production-grade system that identifies change points in complex temporal datasets with minimal latency, achieving 30% reduction in anomaly response time while maintaining interpretability.

Change Point Detection Unsupervised Learning Time-Series Analysis Production Systems

2021

Girasol: A Sky Imaging and Global Solar Irradiance Dataset

Adnan Bashir, et al.

Data In Brief | Elsevier | April 2021

We introduce Girasol, a comprehensive open-source dataset for solar irradiance estimation combining sky imagery with ground-truth measurements. This dataset enables research in computer vision applications for renewable energy forecasting, with applications to grid optimization and distributed solar systems.

Solar Irradiance Computer Vision Open Dataset Renewable Energy
📄 PDF 🔗 ScienceDirect 📊 Dataset Repository
2020

LADPU Smart Meter Data: A Public Dataset for Smart Grid Research

Adnan Bashir, et al.

NSF Public Access Repository | December 2020

This publication describes LADPU, a large-scale smart meter dataset from Los Angeles Department of Public Utilities. The dataset contains consumption patterns from thousands of residential and commercial meters, enabling research in demand forecasting, anomaly detection, and smart grid optimization.

Smart Grid Smart Meters Energy Analytics Demand Forecasting
📄 PDF 🔗 NSF Repository 📊 Data Files
2019

Presidential Elections in Ecuador: Bot Presence in Twitter Networks

Adnan Bashir, et al.

ICEDEG 2019 | International Conference on eDemocracy and eGovernment | April 2019

This study analyzes bot presence in Twitter networks during Ecuador's presidential elections. Using advanced machine learning techniques for bot detection and network analysis, we identify coordinated inauthentic behavior and provide actionable insights for election integrity monitoring and policy recommendations.

Bot Detection Election Integrity Social Media Analysis Network Analysis

2016

Infect-me-not: A User-centric and Site-centric Study of Web-based Malware

Adnan Bashir, et al.

IFIP 2016 | International Federation for Information Processing | May 2016

This paper presents a comprehensive study of web-based malware distribution networks from both user and site perspectives. We develop machine learning models for malware detection and characterize attack patterns, providing insights for improved web security measures and user protection strategies.

Malware Detection Cybersecurity Web Security Machine Learning
