Research Interests

Unsupervised learning, time-series analysis, and production ML systems

Research Vision

My research bridges the gap between cutting-edge machine learning theory and enterprise-scale systems. I focus on developing unsupervised learning methods that can operate on high-dimensional, non-stationary data streams in production environments. Central to my work is the principle that ML systems must be interpretable, efficient, and practical for real-world deployment—not just theoretically novel.

Key themes: unsupervised learning for streaming data, change point detection, explainable AI, LLM applications, and production ML system design.

Core Research Areas

🔄 Unsupervised Learning for Data Streams

Change point detection and regime identification in non-stationary time-series

Developing algorithms to automatically identify significant changes in streaming data without labeled examples. This is critical for anomaly detection, fault identification, and decision support in real-time systems. My work focuses on handling high-dimensional data with computational efficiency suitable for production environments.

Change Point Detection Regime Shift Streaming Data Unsupervised Learning

📊 Time-Series Explainability

Interpretable ML for complex temporal datasets

Creating techniques to explain what ML models learn from time-series data. Understanding why a system detected an anomaly is as important as detecting it. My TSX engine demonstrates how explainability can be integrated seamlessly into production pipelines, enabling stakeholders to trust and act on model outputs.

Explainable AI Interpretability Feature Importance Model Transparency

🤖 LLM-Based State Detection

Applying large language models to identify states in multivariate data

Exploring how foundation models can enhance state identification in data systems. LLMs can reason about complex patterns and provide natural language interpretations of system behavior. This represents a new frontier in bridging symbolic reasoning with modern deep learning for data systems.

LLMs Foundation Models State Detection Natural Language

🏗️ Production ML Systems

Designing scalable architectures for enterprise analytics

Real-world ML systems face challenges beyond algorithms: data governance, compliance, monitoring, and operational reliability. My DRIFT system demonstrates how to build enterprise-grade platforms that handle sensitive data at scale while maintaining security, governance, and real-time performance.

System Design Data Governance MLOps Compliance

🔐 Security & Anomaly Detection

ML for cybersecurity threats and behavioral anomalies

Applying ML to detect malware, insider threats, and adversarial behavior. My background in cybersecurity informs research on how unsupervised techniques can identify zero-day threats and advanced persistent threats that evade signature-based detection.

Malware Detection Threat Intelligence Anomaly Detection Security

📈 Election Integrity & Bot Detection

Large-scale social network analysis for authentic engagement

Continuing research on identifying coordinated inauthentic behavior in social networks. This work has policy implications for election security and information integrity. Developing scalable algorithms that can process millions of interactions while maintaining interpretability for policy makers.

Bot Detection Social Networks Election Integrity Anomaly Detection
